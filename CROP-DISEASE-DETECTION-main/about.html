About This Project

Plant Disease Detection

This application uses deep learning to identify diseases in plants from images of their leaves. Early detection of plant diseases can help farmers take timely action and prevent crop losses.

How It Works

The application uses a convolutional neural network (CNN) trained on the Plant Village dataset, which contains thousands of labeled images of healthy and diseased plant leaves across various species.

Supported Plants and Diseases

The model can identify multiple diseases across different plant species including:

  • Apple (Apple scab, Black rot, Cedar apple rust, Healthy)
  • Blueberry (Healthy)
  • Cherry (Powdery mildew, Healthy)
  • Corn/Maize (Cercospora leaf spot, Common rust, Northern Leaf Blight, Healthy)
  • Grape (Black rot, Esca/Black Measles, Leaf blight, Healthy)
  • Orange (Haunglongbing/Citrus greening)
  • Peach (Bacterial spot, Healthy)
  • Pepper (Bacterial spot, Healthy)
  • Potato (Early blight, Late blight, Healthy)
  • Raspberry (Healthy)
  • Soybean (Healthy)
  • Squash (Powdery mildew)
  • Strawberry (Leaf scorch, Healthy)
  • Tomato (Bacterial spot, Early blight, Late blight, Leaf Mold, Septoria leaf spot, Spider mites, Target Spot, Yellow Leaf Curl Virus, Mosaic virus, Healthy)

Limitations

While this tool can be helpful, it has limitations:

  • It works best with clear, well-lit images of individual leaves
  • The model may not be accurate for plant species or diseases not in its training data
  • This tool should be used as a preliminary guide and not as a replacement for expert consultation

Technology Stack

  • TensorFlow/Keras for the deep learning model
  • Flask for the web application backend
  • HTML/CSS/JavaScript for the frontend
  • Bootstrap for styling
